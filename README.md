# Portfolio
Samsung Software Academy For Youth(SSAFY)에서 교육을 수료 했고, Web 풀스택 개발을 지향하고 있습니다.
Typescript, React, Mobx, Next, Webpack을 활용한 프론트엔드 개발과 Java, Spring boot, Mysql, Express를 활용한 백엔드 개발에 관심이 많습니다.
